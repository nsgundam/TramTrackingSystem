# Specialized Agent Outputs

This directory is reserved for focused Level 2 decisions produced during the Master Refactoring
Roadmap process. Each output should be named `<topic-slug>-agent.md` and must retain the triggering
finding, decision, alternatives, implementation steps, failure modes, migration risk, priority,
difficulty, and unresolved user decisions.

The agent instructions live in [`agents/specialized/`](../../../agents/specialized/README.md). These
files are generated decision briefs, not replacements for the Level 1 audit documents.
